# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability in Party Quips, please report it privately rather than opening a public issue.

**How to report:**
- Email: charles.h.hartmann1@gmail.com
- Subject: `[Security] Party Quips Vulnerability`

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fix (optional)

**Response timeline:**
- Acknowledgment within 48 hours
- Status update within 7 days
- Fix timeline communicated once the issue is confirmed

We will not take legal action against researchers who report vulnerabilities in good faith and follow responsible disclosure.

## Scope

The following are in scope:
- The web app at `chartmann1590.github.io/party-quips`
- The Android APK
- Firebase Realtime Database rules
- Authentication flows

Out of scope:
- Third-party services (Firebase, GitHub)
- Issues requiring physical access to a device
- Social engineering attacks
