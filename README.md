# 🎉 Party Quips

**The hilarious party game show for your phone and TV — no app download required.**

Play together in the same room: one person hosts on a TV or big screen, everyone else joins on their phone. Free, instant, and works on any device.

## How to Play

1. **Open Party Quips on a TV or laptop** → [chartmann1590.github.io/party-quips](https://chartmann1590.github.io/party-quips/)
2. **Pick a game** and a 4-letter room code appears on screen
3. **Everyone opens their phone**, goes to the same URL, and enters the room code
4. **Play!** All answers and voting happen on phones — the results show on the TV

No accounts. No downloads. Works on iPhone, Android, and any web browser.

## Games

### 😂 Witty Quips
Everyone answers the same hilarious prompt, then the group votes for their favorite response. The funniest player wins. Think improv comedy meets party game.

### 🤥 Fib Finder
One answer is true, everyone else makes up a convincing lie. Can you spot the real answer — or fool everyone with your fib?

### 💀 Deadly Trivia
Answer trivia questions correctly or face the consequences. Tense, fast, and brutal for anyone who slept through history class.

## Features

- **Host on any screen** — works on a TV browser, laptop, or desktop
- **Play on any phone** — no app install, just open the link
- **2–8 players** — best with 4–6
- **Free** — no subscriptions, no in-app purchases
- **PWA support** — iPhone users can add it to their home screen
- **Android app** available via APK release

## Self-Hosting / Development

Built with React + TypeScript + Vite + Firebase Realtime Database. Deployed to GitHub Pages.

```bash
git clone https://github.com/chartmann1590/party-quips.git
cd party-quips
npm install
# Add your Firebase config to .env.local (see .env.example)
npm run dev
```
